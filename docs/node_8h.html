enum yuime_children_iterate_return_signal_e
Enumerator
YUIME_CHILDREN_ITERATE_RETURN_SIGNAL_PASS 0 

Iterates children and next siblings.

YUIME_CHILDREN_ITERATE_RETURN_SIGNAL_NO_CHILDREN 1 

Does not iterate children. NOTE: In yuime_node_iterate_children, it does the same effect as YUIME_CHILDREN_ITERATE_RETURN_SIGNAL_NO_SIBLING.

YUIME_CHILDREN_ITERATE_RETURN_SIGNAL_NO_SIBLING 2 

Does not iterate next siblings.

YUIME_CHILDREN_ITERATE_RETURN_SIGNAL_BREAK 3 

Does not iterate children nor next siblings. NOTE: In yuime_node_iterate_children, it does the same effect as YUIME_CHILDREN_ITERATE_RETURN_SIGNAL_NO_SIBLING.
